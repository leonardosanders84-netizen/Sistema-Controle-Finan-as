💸 Sistema de Controle de Finanças Pessoais
> Criado por Leonardo Fagundes Sanders

Sistema completo para organizar receitas e despesas, com cálculos automáticos, gráficos e banco de dados. Desenvolvido com estrutura profissional e layout seguindo as recomendações da ABNT.

---

## 🚀 Funcionalidades
✅ Cadastrar receitas e despesas com categoria, valor, data e descrição
✅ Cálculo automático de total de entradas, saídas e saldo final
✅ Visualização gráfica da distribuição de valores
✅ Excluir lançamentos quando precisar
✅ Dados salvos permanentemente em banco de dados SQLite
✅ Interface clara, organizada e com fontes padronizadas

---

## 🛠️ Tecnologias usadas
- **Python 3**
- **Tkinter** (interface gráfica)
- **SQLite** (banco de dados nativo)
- **Matplotlib** (geração de gráficos)
